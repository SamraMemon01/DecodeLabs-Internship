# Basic Encryption & Decryption Project 

## Overview

This project demonstrates the implementation of a simple Encryption and Decryption system using the Caesar Cipher algorithm. The main objective of this project is to understand the fundamentals of cryptography and how data can be secured by converting readable text into encrypted format and then restoring it back through decryption.

## Features

* Encrypt plain text using Caesar Cipher logic
* Decrypt encrypted text back to original form
* User input based execution
* Demonstrates basic data confidentiality concepts
* Beginner-friendly implementation of cryptography principles

## Technologies Used

* Python
* Basic Cryptography Concepts
* String Manipulation Logic

## How It Works

### Encryption Process

The program takes input text from the user and shifts each alphabet by a fixed number of positions to generate encrypted text.

Example:
HELLO → KHOOR

### Decryption Process

The encrypted text is shifted backward by the same number of positions to restore the original message.

Example:
KHOOR → HELLO

## Example Output

Enter message: HELLO

Encrypted Message: KHOOR

Decrypted Message: HELLO
