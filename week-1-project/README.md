# Week 1 - Project (Password Streangth checker)

## Overview

This project is a Password Strength Checker developed in Python as part of Project 1 during my Cyber Security Internship at DecodeLabs.

The program analyzes a user-entered password and evaluates its strength based on essential security criteria, classifying it as Weak, Medium, or Strong.



## Features

* Checks password length
* Detects uppercase letters
* Detects numeric digits
* Detects special characters/symbols
* Evaluates password strength level
* Displays security feedback to the user



## Technologies Used

* Python
* String Handling
* Conditional Statements
* Loops
* Basic Cyber Security Concepts



## How It Works

The program asks the user to enter a password and checks whether it meets certain security conditions.

The password is evaluated based on:

1. Minimum length requirement
2. Presence of uppercase letters
3. Presence of numbers
4. Presence of special characters

Based on these checks, the password is categorized as:

* Weak Password
* Medium Password
* Strong Password



## Example Output

Input: Hello@123

Output: Strong Password

Input: hello

Output: Weak Password
