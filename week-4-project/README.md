# Cyber Security Project 4 Basic Vulnerability Assessment

## Overview

This project focuses on performing a basic security assessment of a personal computer system in order to identify common security vulnerabilities and understand how small security weaknesses can affect overall system protection.

The purpose of this project was to inspect different security areas of a computer system, identify possible risks, and suggest corrective actions to improve security posture.



## Objectives

The main objectives of this project were:

* To perform a basic vulnerability assessment on a Windows system
* To check important security configurations of the computer
* To identify security weaknesses that may expose the system to cyber threats
* To understand practical cybersecurity auditing techniques
* To recommend security improvements based on findings



## Security Checks Performed

The following system security checks were performed during this assessment:

### 1. Firewall Configuration Check

The system firewall settings were reviewed through Windows Security settings to verify whether firewall protection was enabled.

**Result:** Firewall was found enabled.



### 2. Windows Update Check

The operating system update section was inspected to verify whether the system had the latest security patches installed.

**Result:** Multiple Windows updates were pending.



### 3. Account Security Check

Google account security settings were reviewed to check whether Two-Step Verification was enabled.

**Result:** Two-Step Verification was disabled.



## Vulnerabilities Identified

The assessment identified the following security weaknesses:

* Pending operating system security updates
* Two-Step Verification disabled on user account

These weaknesses can increase the chances of unauthorized access or exploitation of known vulnerabilities.



## Security Recommendations

Based on the assessment, the following actions are recommended:

* Install all pending Windows updates regularly
* Enable Two-Step Verification for important online accounts
* Keep firewall protection enabled permanently
* Regularly monitor system security settings



## Tools Used

The following tools/settings were used during the assessment:

* Windows 10 Security Settings
* Windows Update Settings
* Google Account Security Settings



## Conclusion

This project provided practical understanding of how basic system security auditing works. Even simple security checks can help identify weaknesses that may lead to larger cybersecurity risks if left unresolved.

Regular security monitoring and timely corrective actions are important for maintaining a secure computing environment.
